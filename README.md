# Binaural Hearing Aids using Arithmetic Fourier Transforms

## Overview
This project explores the design of a binaural hearing aid system that preserves spatial hearing cues (interaural time, level, and phase differences) while minimizing signal processing delay.

## Motivation
Conventional digital hearing aids often rely on FFT-based processing pipelines that introduce perceivable delays, which can degrade auditory localization.

Prior research has shown:
- High-delay processing negatively impacts spatial hearing
- Lower-delay systems improve localization performance over time

This project investigates alternative signal processing approaches to address these limitations.

## Approach
The system explores arithmetic-transform-based signal processing methods as low-latency alternatives to FFT-based pipelines.

The goal is to reduce computational delay while preserving interaural cues.

## Research Context
This work connects ideas from:
- Analytic number theory
- Signal processing
- Transducer design
- Psychoacoustics
- Audiology
